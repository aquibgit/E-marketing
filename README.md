# E-Market (Django E-Commerce Platform)

A full-stack e-commerce web application built using Django, featuring product management, cart functionality, and order tracking.

## Features
- User authentication and session management
- Product listing and dynamic CRUD operations
- Shopping cart and checkout system
- Admin dashboard for product and order management

## Tech Stack
- Backend: Django
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite / MySQL

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run server:
   python manage.py runserver
